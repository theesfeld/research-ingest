research-ingest

Local-first research ingest for Obsidian. Browser send, vault watch, Grok SuperGrok session processing. No pay-per-token xAI API by default.

GitHub Phase tracker #1

Install

git clone https://github.com/theesfeld/research-ingest.git
cd research-ingest
cargo install --path crates/research-ingest
cargo install --path crates/research-send
cargo install --path crates/research-mcp
research-ingest enable
# Daily: Brave Send only (extension + hotkey)

AI path

Language work uses your logged-in grok CLI session (SuperGrok). The tools remove XAI_API_KEY from the child environment so session auth wins.

OCR and transcript

Images use tesseract OCR. Audio and video use ffmpeg plus whisper.cpp for automatic transcripts before Grok writes project notes.

Vault layout

Obsidian Vault/
  raw/incoming/     ← drop or browser send
  raw/processed/
  raw/extracts/
  wiki/projects/<slug>/
  index.md
  map-of-content.md
